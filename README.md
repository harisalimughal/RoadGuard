# RoadGuard

**Smarter roads, safer journeys.**  
AI-powered traffic safety & alert system — predictive alerts, incident reporting, safe routes, and emergency SOS.

Final Year Project — Department of Computer Science, International Islamic University Islamabad.

## Project structure

- **`frontend/`** — React + Vite dashboard (web app)

## Run the frontend

```bash
cd frontend
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173). See `frontend/README.md` for more details.
